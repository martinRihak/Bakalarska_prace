# Webová aplikace pro sběr, zpracování a vizualizaci dat ze senzorů

## Popis projektu

Tento projekt je webová aplikace určená pro sběr, zpracování a vizualizaci dat ze senzorů pomocí sběrnice RS-485 na Raspberry Pi. Cílem je vytvořit robustní systém, který umožní efektivní správu a analýzu naměřených dat.

## Hlavní funkce

- Komunikace se senzory přes RS-485

- Zpracování a ukládání dat do databáze

- Webové rozhraní pro vizualizaci dat

- Možnost exportu dat
